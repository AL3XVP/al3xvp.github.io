About me

My name is Alexander Phan, and I'm pursuing a Bachelor of Science in Computer Science at the University of Massachusetts Boston.

I like building software that people actually reach for — tools that feel fast, look clean, and hold up under real use. My coursework runs deep on fundamentals (data structures, computer architecture, compilers, operating systems), and outside the classroom I ship full-stack web apps with Next.js, React, and TypeScript.

I care about the details others skip: accessibility, security, and the small interactions that make an interface feel considered. When I'm not coding, I'm usually designing, mentoring, or figuring out how to make a messy process a little simpler.

Featured Project

SSS Web — Student Support Services Microsite

A modern, fully responsive microsite for a university Student Support Services program, inspired by the UMass Boston SSS page. Built data-first: all content — services, eligibility, staff, hours, FAQ — lives in typed config files, so the site can be updated without touching a single component.

  • Data-driven architecture — edit content in /data/*.ts, no code changes
  • Accessibility-first: semantic landmarks, skip links, visible focus, keyboard FAQ
  • Mobile-first responsive layout with a custom "beacon-blue" design system
  • Next.js 14
  • React 18
  • TypeScript
  • Tailwind CSS

Featured Project

Attendance App — QR Classroom Check-In

A QR-based classroom attendance system for instructors. Students scan a rotating code to check in; the dashboard scores Present / Tardy / Absent automatically from time-since-start, supports multiple class sections, and exports clean rosters and summaries to CSV or Excel.

  • Security-first: constant-time login, signed httpOnly sessions, rate limiting
  • Student PII (email & phone) encrypted at rest and masked in the UI
  • libSQL/Turso in production, auto on-disk SQLite for local dev; CSV/Excel exports
  • Next.js 15
  • React 19
  • libSQL / Turso
  • ExcelJS
  • Tailwind CSS
